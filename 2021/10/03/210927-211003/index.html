크롤링, 셀레니움 with Python

  • 동적 크롤링 : 쇼핑몰, 주가정보 등 한 주소창에 여러 페이지로 이루어진 사이트에서 필요한 정보 가져오기
  • 셀레니움 : 구글 드라이브를 사용해서, 각종 사이트 로그인, 원하는 작동방식을 설정하여 자동으로 데이터 수집하기
  • 토이프로젝트 : 프로젝트 아이디어 기획 - 데이터셋 확보 - 데이터셋 활용여부 검증 - 시각화 - 발표 과정 수행 중

정리

  • 크롤링, 셀레니움 : robot.txt 등을 사용하여, 학습용도 외에 이후 실전에서 사용할때, 정보 윤리적인 문제가 생기지 않게 꼼꼼히 확인해보는 습관을 가져야한다
  • 토이프로젝트 : 아이디어 설정, 원하는 인사이트를 얻기 위한 데이터 확보가 생각보다 굉장히 어렵고 많은 시간을 투자해야 한다는 것을 체감하는 중이다.
    ( 나는 데이터 전처리에 좀 강점이 있는것 같다, 아이디어와 데이터 검증은 많이 미숙한편))

개선사항

  • 주가별 5년치 데이터 수집 / 쇼핑몰 best상품 크롤링 코드 제작으로 크롤링, 동적크롤링 연습
  • html에 대한 학습은 아직까진 큰 필요성을 느끼지 못하고 있다- 커리어 미팅 대비는 프로젝트 이후로 좀 미뤄야겠다

차주 보완사항

  • 토이프로젝트를 마무리하면서, 프로젝트 피드백을 해보자
  • 데이터셋 확보를 할 수 있을 만한 사이트, 정보를 미리 확보해보자
  • 위의 방법으로도 얻기 어려운 정보들은 지금처럼 크롤링으로 데이터셋을 확보해놓자. 언젠간 써먹겠지(코드든 내용이든)
